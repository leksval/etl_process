Instalation

1.1. Download files from Github repository:
https://github.com/leksval/etl_process

1.2. Then install packages. File with required packages: /etl_process/requirements.txt
pip install ./etl_process/requirements.txt

1.3. Then to set up SQL in terminal run coommand:
docker run --name etl_imdb -e POSTGRES_PASSWORD=postgres -e POSTGRES_USER=postgres -e POSTGRES_DB=imdb -d -p 5432:5432 postgres

1.4. Run main file in terminal:
python wsgi.py

1.5 Use link generated in terminal to launch app in browser.
